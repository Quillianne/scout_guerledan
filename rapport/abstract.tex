\section*{Abstract}
This project addresses cooperative localization for a three-boat fleet operating on Lake Guerlédan, with one mother boat and two scout boats. The objective is to estimate consistent position sets under uncertainty rather than a single point estimate. To support field operations, a unified Python communication library was developed to standardize control and telemetry access across all boats. A web interface was also implemented to centralize supervision, mission control, interval-based observation, and replay analysis.

The methodology combines dead-reckoning prediction with interval contraction using inter-boat distance constraints and GPS bounding boxes. A first campaign focused on command validation and software integration, while a second campaign at Guerlédan enabled real-world measurements and interval-localization experiments. Results show that the proposed architecture is operational for multi-boat supervision and robust interval observation, while also highlighting remaining work on closing the loop with fully interval-driven navigation control.

\noindent\textbf{Keywords:} cooperative localization, interval analysis, BlueBoat, MAVLink, web supervision, field robotics.
