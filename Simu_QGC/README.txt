- Télécherger Ardurover via https://firmware.ardupilot.org/Rover/stable-4.2.3/SITL_x86_64_linux_gnu/

- Faire une copie de l'exécutable dans chaque fichier "Ardurover X"

- Télécharger QGroundControl via https://docs.qgroundcontrol.com/Stable_V5.0/en/qgc-user-guide/getting_started/download_and_install.html

- Copier/déplacer l'exécutable dans le fichier Simu_QGC

- Lancer run_multiboat.sh via le terminal ou en tant qu'application
